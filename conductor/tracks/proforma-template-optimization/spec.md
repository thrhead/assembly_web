# Spec: Enterprise Proforma & Reporting Optimization

## Requirements
- Professional, high-resolution PDF generation.
- Correct handling of multi-page data tables (auto-pagination).
- Customizable headers/footers with company branding.
- Secure URLs for document downloads.

## Technical Constraints
- PDF generation must handle Turkish characters correctly (font embedding).
- Files must be optimized for small size without losing quality.

## Success Criteria
- [ ] Proforma invoices include all required commercial information (Logo, Tax ID, Legal Text).
- [ ] Reports are generated automatically and stored correctly.
- [ ] Bulk export functionality supports ZIP or combined PDF downloads.
