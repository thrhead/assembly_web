# Specification: UI/UX Accessibility & Touch Standardization

## Goal
Standardize global accessibility and touch interactions across web and mobile platforms based on the `ui-ux-pro-max` audit recommendations.

## Scope
1.  **Web Accessibility:** Correct mobile font sizes and restore browser zoom functionality.
2.  **Mobile Touch Interaction:** Standardize touch target sizes and interaction feedback.
3.  **Visual Consistency:** Align branding colors across themes and improve interactivity cues.

## Requirements
- Minimum font size of 16px on mobile (Web).
- Minimum touch target size of 44x44px for interactive elements (Mobile).
- Visible and consistent primary color across Light and Dark modes (Mobile).
- Standardized `cursor-pointer` for all clickable elements (Web).
